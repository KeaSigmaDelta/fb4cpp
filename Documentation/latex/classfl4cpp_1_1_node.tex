\subsection{Detailed Description}
Base class for all classes representing Graph Api nodes. See\+: \href{https://developers.facebook.com/docs/graph-api/quickstart/v2.0}{\tt https\+://developers.\+facebook.\+com/docs/graph-\/api/quickstart/v2.\+0}

N\+O\+TE\+: Setter methods of the \hyperlink{classfl4cpp_1_1_node}{Node} class are used for initialization of the local copy, and has no affect on Graph Api copy.
