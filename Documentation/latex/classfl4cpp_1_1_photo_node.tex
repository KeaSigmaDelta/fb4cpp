\subsection{Detailed Description}
Represents Photo \hyperlink{classfl4cpp_1_1_node}{Node} of the facebook Graph Api. You can find detailed description of the fields and edges of this node in the Graph Api documentation. See\+: \href{https://developers.facebook.com/docs/graph-api/reference/v2.0/photo}{\tt https\+://developers.\+facebook.\+com/docs/graph-\/api/reference/v2.\+0/photo}

N\+O\+TE\+: Setter methods of the \hyperlink{classfl4cpp_1_1_photo_node}{Photo\+Node} class are used for initialization of the local copy, and has no affect on Graph Api copy.
