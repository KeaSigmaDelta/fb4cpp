\subsection{Detailed Description}

See\+: \href{https://developers.facebook.com/docs/graph-api/reference/v2.0/album}{\tt https\+://developers.\+facebook.\+com/docs/graph-\/api/reference/v2.\+0/album}

N\+O\+TE\+: Setter methods of the \hyperlink{classfl4cpp_1_1_album_node}{Album\+Node} class are used for initialization of the local copy, and has no affect on Graph Api copy.
