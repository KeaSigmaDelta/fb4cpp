\subsection{Detailed Description}
Contains string values which is used by fl4cpp library for interaction with facebook Graph A\+PI.

App Id value should be set by the programmer before using any \hyperlink{classfl4cpp_1_1_session}{Session} class methods.

User Agent value depends on which browser engine is used for a login sequence.

Graph Api U\+RL should keep its default value, unless Facebook changes its api.
