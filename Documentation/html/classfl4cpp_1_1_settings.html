Detailed Description

Contains string values which is used by fl4cpp library for interaction with facebook Graph API.

App Id value should be set by the programmer before using any Session class methods.

User Agent value depends on which browser engine is used for a login sequence.

Graph Api URL should keep its default value, unless Facebook changes its api.
