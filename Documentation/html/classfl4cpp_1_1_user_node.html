Detailed Description

Represents User Node of the facebook Graph Api. You can find detailed description of the fields and edges of this node in the Graph Api documentation. See: https://developers.facebook.com/docs/graph-api/reference/v2.0/user

NOTE: Setter methods of the UserNode class are used for initialization of the local copy, and has no affect on Graph Api copy.
