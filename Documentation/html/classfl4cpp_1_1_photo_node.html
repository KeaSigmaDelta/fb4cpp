Detailed Description

Represents Photo Node of the facebook Graph Api. You can find detailed description of the fields and edges of this node in the Graph Api documentation. See: https://developers.facebook.com/docs/graph-api/reference/v2.0/photo

NOTE: Setter methods of the PhotoNode class are used for initialization of the local copy, and has no affect on Graph Api copy.
